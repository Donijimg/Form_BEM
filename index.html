Validaciones JS

Los campos se validaran de la siguiente manera

  • Ficha:

    se puede ingresar cualquier caracter

  • Nombre:

    solo se puede ingresar letras

  • Apellidos:

    solo se puede ingresar letras

  • Correo:

    solo se puede ingresar Correo valido

  • Telefono:

    solo se puede ingresar numeros

  • Genero:

    solo se puede seleccionar un Genero

  • Ciudad:

    solo se puede seleccionar una Ciudad

  • Habilidades:

    solo se puede seleccionar como minimo 3 Habilidades
